Roll-Back DC Guide

Restore the previous Discord UI!

1. Download and Install Vencord

Head over to the Vencord website, download and install it:

Go to Vencord

2. Enable the Experiments Plugin

Once you've installed Vencord, go to the settings inside Vencord and enable the "Experiments" plugin to activate the functionality:

  • Open the Vencord settings
  • Navigate to the 'Plugins' section and enable 'Experiments'.

3. Activate Desktop Visual Refresh in Discord

In Discord:

  • Open `Settings`
  • Scroll down to the "DEVELOPERS ONLY" section.
  • Click the first setting (Experiments) and type desktop refresh in the box.

4. Restore Classic UI

Click the dark box and select the option called Treatment 0: Control to restore the classic Discord UI and enable Vencord theme functionality.
